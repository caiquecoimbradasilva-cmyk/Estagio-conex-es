# Estagio-conex-es
Site de estagio, formando conexões com empresas e futuros melhores.

TECNOLIGIAS UTILIZADAS:

HTML (HyperText Markup Language): estruturação do conteúdo e das páginas.
CSS (Cascading Style Sheets): estilização e responsividade visual do site.
JavaScript: interação e dinamismo na navegação entre as páginas.
